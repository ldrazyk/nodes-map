# Nodes Map GUI

## Actions

### Selecting Node

1. Select
    * Click Node on svg map
    * Select from options in Menu
2. Show Nodes Info in Menu

### Point at selected Node

* Draw marker around node
* Zoom to node

## Elements

### Selected Node Info

Showing:

* image
* id
* label
* links
* other
  
Not showing:

* size
* x, y
